README para facilitar a compreensão:

	Os arquivos FCFS e SCOREs tem os dados brutos até a linha 2786, em baixo delas possuem dados analisados em cima desses brutos, como wt médio, score, entre outros.

	No arquivo FCFS é onde gero o histograma de score, fica abaixo dos outros dados anotados.

	O outro arquivo ("Análise...") é o principal, lá tenho as anotações (a mão) em cima dos dados obtidos nos outros arquivos e é onde gero os gráficos, e que estão sempre ao lado direito dos dados anotados.

	Lembrando que o que tive dificuldade para gerar e seria bom é um gráfico na mesma forma dos que analisam 3 usuários e possuem o wt médio (sem wt 0), porém com max, min e desvio padrão em conjunto deles.
Algo assim ou próximo facilitaria para interpretar o script e aprender como fazer.
